&nbsp;ISSUEFOLIO



ISSUEFOLIO is a full-stack web application that recommends

relevant GitHub issues based on user repositories, language

usage, and repository activity.



Tech Stack

\- Backend: Python, Flask

\- Frontend: React

\- Database: PostgreSQL

\- Async: Celery, Redis

\- APIs: GitHub REST \& GraphQL

\- Docker (planned)



&nbsp;Features

\- GitHub OAuth authentication

\- Fetch repositories and language usage

\- Issue recommendation using activity scoring

\- Background jobs for periodic refresh



Status: MVP in progress
